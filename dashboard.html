How it works

Click a Learn button, then point your heater's original remote at the IR receiver on the device and press the corresponding button within 5 seconds. The code is stored permanently — it survives reboots and works for any brand or protocol. If a code is already learned it will be replaced. If no code is stored, the button will have no effect until learned.
