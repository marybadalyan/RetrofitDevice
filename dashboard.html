How it works

Click a Learn button, then point your heater's original remote at the IR receiver on the device and press the corresponding button within 10 seconds. The code is stored permanently — it survives reboots and works for any brand or protocol. If a code is already learned it will be replaced. The fallback (built-in NEC) is used when no code is stored.
